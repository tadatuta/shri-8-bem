# shri-8-bem

### [Demo](https://bem-shri.herokuapp.com/)

Три конечные сборки - под mobile, tablet, desktop.  
Определение user agent происходит на сервере, и в зависимости от этого отдаётся нужный бандл.

### Установка
```sh
npm install
```

### Запуск сервера
```sh
npm start
```

### Запуск (development)
```sh
npm run dev
```

### Production сборка
```sh
npm run build
```

### Линтер
```sh
npm run lint
```
